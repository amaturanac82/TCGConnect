Servidor base con Node.js y Express

Esta aplicación sirve contenido web, utiliza rutas públicas y registra accesos en un archivo plano.

Ver estado del servidor

Rutas disponibles

  • / - Página principal.
  • /about - Información del proyecto.
  • /status - Respuesta JSON del servidor.
  • /visitas - Registro de accesos.

Objetivo del proyecto

Construir la base de una aplicación backend con estructura modular, contenido estático y persistencia simple en archivo plano.

Tecnologías usadas

  • Node.js
  • Express
  • dotenv
  • nodemon
